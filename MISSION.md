# PaperLesson 项目使命

## 愿景

打造一个**精读导向**的个人课程网站，每一页课程都是对一篇论文或一本教材的深度拆解。

## 核心理念

- **少而精**：不追求课程数量，追求每一课都能让学习者真正掌握
- **可交互**：每课包含可运行的代码、折叠练习、思考题
- **层层递进**：从直觉到公式，从代码到理论，逐步深入
- **学必有获**：每节课都有明确的学习目标和验证环节

## 学习者画像

这个网站为以下读者设计：

- 研究生：需要系统性地精读本领域经典论文
- 自学者：希望从零搭建领域知识体系
- 研究者：快速回顾和对比不同方法论

## 内容原则

1. 每篇论文/每本教材拆解成 3-6 节可连续学习的课程
2. 公式必须配解读，代码必须可运行
3. 每节课末有练习，学完即检验
4. 保留完整的引用和延伸阅读

## 技术原则

- 纯静态网站，无需后端，直接通过 GitHub Pages 部署
- 数据驱动：新增课程只需编辑 `data/course-catalog.js`，无需修改 `index.html`
- 完全离线可用（除 CDN 字体和 MathJax）
